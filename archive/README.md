- 過去の開発段階で使用されたテストファイル
- 特定機能の動作確認やバグ修正の確認に使用
- 現在は不要だが、将来的に参考になる可能性がある

### reports/
- 過去の作業完了レポート
- プロジェクト引き継ぎ書
- テストガイド・シナリオ

### session_html/
- 過去のセッション結果HTMLレポート
- セッション実行の記録
